# arrimo-pro-challenge
A famous tech blogger would like to have an API where readers can sign up for his weekly newsletter, the blog owner also want to be able to fetch the list of interested readers.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
